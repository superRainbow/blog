    撒花～部落格增加留言功能囉

    金魚腦 2021/4/30 jsblog

    # 前言

    之前有想過要整合留言功能， 那時看了方式有串第三方平台與使用 github issue， 不過想說不知道文章會寫到何時～先擱著(看雖自己)， 但現在覺得好像可以加進來看看惹！感覺今年的我好像是真的想認真寫文章 XD 接下來就來看我的步驟吧～

    # 選擇

    基於我的部落格是將文章都放在 github 上， 最好的方式就是都統一放在 github 中， 因此我就不考慮 Disqus 或是 LeanCloud 惹， 選擇 Vssue 來實現留言

    # Vssue 介紹

    Vssue 是利用平台提供的 OAuth API 使得用戶可以登錄並發表評論， 以下是大致的流程：

    來源：Vssue 官網 (opens new window)

    # 導入

    預設 vuepress-theme-reco 就有留言功能，我們只需要去 config.js 中加入設定即可

    # 1. 申請 github OAuth 2

    申請 github OAuth (opens new window) 來取得 clientId 與 clientSecret

    1. Register a new OAuth application (opens new window)
    2. 將 Homepage URL 和 Authorization callback URL 設置為你的網站 URL
    3. 完成後，即可獲取 Client ID 和 Secret

    # 2. config.js 設定

    vssueConfig: {
    platform: 'github',
    owner: 'superRainbow',
    repo: 'blog',
    clientId: process.env.CLIENT_ID,         // 使用環境變數去管理
    clientSecret: process.env.CLIENT_SECRET, // 使用環境變數去管理
    locale: 'en'    // 由於 vssue 的語系中文只有簡體，所以我設定為英文顯示
}
