    git init 之後勒？

    金魚腦 2021/4/14 git

    # 前言

    平常我們常使用 git init 來版控專案，
    但我們有沒有好好思考指令底下做了哪些事情呢？

    這篇來好好瞭解一下 git init 吧～

    # git init

    # 使用時機

    要將某資料夾加入版控讓 git 追蹤時使用

    # 指令

    git init [--options] [directory]
    1

    # 實作

    # 建立一個專案 & git 初始化

    # 建立一個專案
> mkdir try-git && cd $_

# git 初始化
> git init
# Initialized empty Git repository in /try-git/.git/
